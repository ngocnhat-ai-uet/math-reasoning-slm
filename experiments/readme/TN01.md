## generations.jsonl (output thô của model)

Chức năng: Để đọc output model. Ví dụ
{
  "run_id": "tn01_gemma4e2b_base_bespoke_concise_pv1_dv1",
  "dataset": "bespoke",
  "question_idx": 1001,
  "question": "Simplify ...",
  "hint": "Rewrite radicals using fractional exponents.",
  "label": "3/4",
  "model_output": "Use fractional exponents... \\boxed{3/4}",
  "output_token_length": 590
  "finish_reason": "stop"
}

## predictions.parquet (output đã được chấm đúng sai)
 
Chức năng: extract answer, tính accuracy, thống kê. Schema
- run_id: tn01_gemma4e2b_base_bespoke_concise_pv1_dv1
- question_idx: 1001
- extracted_answer: 3/4
- label: 3/4
- is_correct: True

}
